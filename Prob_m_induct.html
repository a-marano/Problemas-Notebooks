Problemas máquina inducción¶

Problema 1¶

Un motor de induccin trifsico de gran potencia est alimentado desde una red de media tensin (MT) a travs de un transformador. Las caractersticas nominales del motor (datos de la placa de caractersticas) son:

  • Velocidad nominal: $n_M = 2930$ rpm
  • Potencia nominal: $ P_u = 75$ kW
  • Rendimiento: $\eta = 86.27\%$
  • Tensin nominal: $U_n=400$ V
  • Factor de potencia nominal: fp = 0.92

Por su parte, los datos del transformador son:

  • Potencia nominal: $S_n=125$ kVA
  • 20/0.4 kV Yy0
  • Datos de cc: $\epsilon_{cc}=5\%$, $\cos\varphi_{cc}=0.2$
  • Datos de vaco: $P_0=5$ kW, $\cos\varphi_0=0.1$

Determinar, para el punto de funcionamiento nominal del motor:

  1. Potencias activa y reactiva absorbidas por el motor
  2. Par mecnico til del motor
  3. ndice de carga del transformador
  4. Regulacion del transformador
  5. Rendimiento del transformador

# Solucin problema 1

n_m = 2930 # rpm
Pu = 75.0e3 # W
rto_m = 0.8627
Un = 400 # V
fp_m = 0.92

S_t = 125.0e3 #kVA
U1n, U2n = 20.0e3, 400.0
e_cc, fp_cc = 0.05, 0.2
P0, fp_0 = 5.0e3, 0.1

import numpy as np

Pe = Pu/rto_m
Qe = Pe/fp_m*np.sin(np.arccos(fp_m))

Mu = Pe/(n_m*2*np.pi/60)

# el ndice de carga expresado como la relacin entre la S2 y la S nominal del trafo
c = np.sqrt(Pe**2 + Qe**2)/S_t

reg = c*e_cc*np.cos(np.arccos(fp_cc)-np.arccos(fp_m))

# para el rendimiento es necesario conocer la potencia del ensayo de cc (Pcc):
# rto = P2 / (P2+P0+c**2*Pcc)
# Pcc se puede obtener a partir de los datos de cc del enunciado:
# --> condiciones ensayo cc: alimentacin a Ucc con corriente I2n: 
# --> |Zcc| = |Ucc_fase|/|I2n|
I2n = S_t/U2n/np.sqrt(3)
Zcc = (e_cc*U2n/np.sqrt(3))/I2n
Rcc = Zcc*fp_cc
Pcc = 3*Rcc*I2n**2
rto = Pe/(Pe + P0 + c**2*Pcc)

Problema 2¶

Un motor de induccin trifsico conectado en estrella a una red de 280 V y 50 Hz tiene unas prdidas en el hierro de 280 W. A plena carga desarrolla un par til de 44 Nm girando a 710 rpm. En este rgimen de carga la potencia en el entrehierro es de 3500 W.

En esas condiciones, determinar:

  1. Potencia en el eje del motor
  2. Deslizamiento con el que trabaja el motor
  3. Resistencia estatrica sabiendo que la potencia absorbida de la red es de 2870 W con un factor de potencia de 0.82
  4. Prdidas mecnicas y prdidas Joule en el rotor
  5. Rendimiento del motor

# Solucin 2
#---
import numpy as np

U = 380.0 # V
Tu = 44.0 # Nm
n = 710.0 # rpm
Pe, cos_phi = 3870.0, 0.82
Pa = 3500.0 # W
Pfe = 280.0 # W

Pu = Tu*(n*2*np.pi/60)
# la velocidad de sincronismo tiene que ser de 750 rpm (p=4)
n_s = 750
s = (n_s-n)/n_s
# Las prdidas Joule se pueden calcular de un balance de potencias:
# Pe = Pje + Pfe + Pa.
# Pje = 3*Re*Ie**2
# Ie puede obtenerse de la potencia elctrica absorbida por el motor:
# Pe = sqrt(3)*U*Ie*cos(phi)
Pje = Pe - Pfe - Pa
Ie = Pe/(np.sqrt(3)*U*cos_phi)
Re = Pje/3/Ie**2

# prdidas Joule en el rotor: Pjr = s*Pa
Pjr = s*Pa
# prdidas mecnicas:
Pm = Pa-Pu-Pjr
# rendimiento
rto = Pu/Pe


print "SOLUCIN 2"
print "Pu = {:.3f} kW".format(Pu/1000)
print "s = {:.3f} %".format(s*100)
print "Re = {:.2f} ".format(Re)
print "Pjr = {:.1f} W".format(Pjr)
print "Pm = {:.1f} W".format(Pm)
print "Pjr = {:.1f} W".format(Pjr)
print "rto = {:.3f} W".format(rto)

Problema 3¶

En un equipo motobomba (grupo hidrulico) el motor es asncronotrifsico con esttor hexapolar y conexin en estrella. Su tensin nominal es de 400 V y se sabe que tiene una resistencia estatrica de 1.2 y unas prdidas en el hierro de 187 W. Girando a 965 rpm el motor absorbe una potencia de 5.25 kW estando alimentado a su tensin nominal y acciona una bomba que posee un par resistente cuadrtico ($M_R$) que responde a la ecuacin indicada a continuacin:

$$M_R = 0.0018 \Omega^2 +0.19 \Omega +8$$

determinada usando unidades del sistema internacional.

Para esas condiciones de funcionamiento, se pide:

  1. Par resistente
  2. Deslizamiento del motor
  3. Potencia mecnica interna suponiendo unas prdidas mecnicas de 50 W
  4. Prdidas joule en el rotor.
  5. Intensidad absorbida por el motor
  6. Rendimiento del motor

# Solucin 3

import numpy as np

U = 400.0 # V
Re = 1.2 # 
Pfe = 187.0 # W
w = 965.0 # rpm
Pe = 5.25e3 # W
ns = 1000.0 # rpm (p=3)
Pm = 50.0 # W

def mr_bomba(omega=w,a=0.0018, b=0.19, c=8):
    omega = omega*2*np.pi/60
    return a*omega**2 + b*omega + c

# el par resistente es:
Mr = mr_bomba(w)

# deslizamiento:
s = (ns-w)/ns

# potencia mecnica interna Pmi = Pu + Pm
Pu = Mr*(w*2*np.pi/60)
Pmi = Pu + Pm

# prdidas joule rotor: Pjr = s*Pa = s*(Pmi+Pjr)
Pjr = s/(1-s)*Pmi

# Pe = Pa + Pje + Pfe --> Pje
# Pje = 3*Re*Ie**2 --> Ie

Pje = Pe - Pjr/s - Pfe
Ie = np.sqrt(Pje/3/Re)

rto = Pu/Pe

print "Solucin 3"
print "Mr = {:.2f} Nm".format(Mr)
print "s = {:.3f}".format(s)
print "Pmi = {:.1f} W".format(Pmi)
print "Pjr = {:.1f} W".format(Pjr)
print "Ie = {:.2f} A".format(Ie)
print "rto = {:.3f}".format(rto)
